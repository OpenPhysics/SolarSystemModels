# Credits — SolarSystemModels

SceneryStack port of the NAAP 'Solar System Models' lab. Compare the Earth-centered Ptolemaic model (deferent and epicycle) with the Sun-centered system and the planetary configurations it produces.

## Upstream inspiration

This simulation is inspired by materials from the
[Nebraska Astronomy Applet Project (NAAP)](https://astro.unl.edu/naap/).
It is an independent [SceneryStack](https://scenerystack.org/) reimplementation, not an official NAAP product.
